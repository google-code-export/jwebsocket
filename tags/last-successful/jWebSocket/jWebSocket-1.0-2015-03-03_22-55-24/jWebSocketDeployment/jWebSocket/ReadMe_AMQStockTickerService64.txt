Please copy jWebSocketAMQStockTickerService64.exe to the /bin folder below your jWebSocket home folder and
use the jWebSocketInstallAMQStockTickerService64.bat to install the service in your 64bit Windows environment.
Please mind that a running Apache ActiveMQ instance is required to run this service!
To start and stop the service please use the "services" dialog of Windows.
To uninstall the service please use the jWebSocketUninstallAMQStockTickerService64.bat.
